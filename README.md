# 🍕 Pizza Sales Dashboard | Power BI

## 📌 Project Overview

The Pizza Sales Dashboard is an interactive business intelligence solution built using **Microsoft Power BI**. This project analyzes pizza sales data to provide meaningful insights into business performance through dynamic visualizations and KPIs.

The dashboard helps identify sales trends, top and bottom-performing products, customer ordering patterns, and revenue distribution across pizza categories and sizes.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Track key business KPIs.
- Identify top and worst-performing pizzas.
- Understand customer ordering behavior.
- Analyze sales by pizza category and size.
- Monitor monthly revenue trends.
- Build an interactive and user-friendly dashboard.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query

- Data Modeling

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)

- 💰 Total Revenue
- 🛒 Total Orders
- 🍕 Total Pizzas Sold
- 💵 Average Order Value
- 📦 Average Pizzas per Order

### Visualizations

- Monthly Revenue Trend
- Sales Distribution by Pizza Category
- Sales Distribution by Pizza Size
- Orders by Day of Week
- Total Pizzas Sold by Size & Category
- Revenue by Pizza Name (Treemap)
- Top Pizza by Revenue
- Top Pizza by Orders
- Top Pizza by Quantity
- Worst Pizza by Revenue
- Worst Pizza by Orders
- Worst Pizza by Quantity

### Interactive Filters

Users can filter the dashboard using:

- Month
- Pizza Category
- Pizza Size
- Reset Filters Button

## 📈 Key Business Insights

- Total Revenue: **818K**
- Total Orders: **21K**
- Total Pizzas Sold: **50K**
- Average Order Value: **38.32**
- Average Pizzas per Order: **2.32**
- Highest Revenue Pizza: **The Thai Chicken Pizza**
- Highest Ordered Pizza: **The Classic Deluxe Pizza**
- Lowest Performing Pizza: **The Brie Carre Pizza**
- Highest Monthly Revenue: **July**
- Highest Number of Orders: **Thursday**
- Large-sized pizzas generated the highest sales contribution.
